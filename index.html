Qui suis-je ?

Le code est plus qu'un métier, c'est une passion.

Passionné par l'informatique depuis mon plus jeune âge, je suis un développeur et architecte polyvalent. Mon parcours est guidé par une curiosité insatiable qui m'a poussé à explorer toutes les couches de la technologie, de la programmation bas niveau (C, ASM), à la création d'interfaces web ultra-modernes (React).

C'est cette volonté qui m'a amené à concevoir ZenitDB, mon propre SGBD, pour maîtriser le stockage de données à un niveau fondamental.

Créateur de Plasma AI Hermes, je fine-tune des modèles de langage et développe des bots (WhatsApp sous Termux). Mon objectif est de fusionner l'IA avec des architectures logicielles robustes.

Mon Arsenal Technique

Du bas niveau jusqu'au cloud, voici les technologies que je maîtrise.

Python & Intelligence Artificielle

Création et entraînement de modèles de langage (LLM) basés sur l'architecture Hermes 3B. Développement de bots automatisés complexes pour diverses plateformes (Gemini, WhatsApp). Maîtrise des scripts d'automatisation avancés et du traitement de données massif.

JavaScript / TypeScript / HTML / CSS

Développement web frontend et backend dynamique. Conception d'interfaces utilisateur modernes (Glassmorphism, animations avancées), gestion de la logique asynchrone complexe et intégration d'API en temps réel. Création d'expériences web immersives.

React & Vite

Développement de Single Page Applications (SPA) hautement performantes. Architecture de composants réutilisables, gestion d'états globaux et optimisation des temps de chargement pour des plateformes comme Cine-Lux.

C / C++ & Architecture de Systèmes

Programmation système bas niveau, gestion manuelle de la mémoire (pointeurs, allocation) et optimisation extrême. Conception d'architectures logicielles complexes comme le moteur de base de données ZenitDB.

Node.js & Docker (DevOps)

Création de serveurs backend robustes en Node.js. Conteneurisation des applications avec Docker pour garantir des environnements de développement et de production iso-fonctionnels. Gestion de microservices.

Assembleur (ASM)

Compréhension intime du fonctionnement des processeurs. Écriture d'instructions directement compréhensibles par la machine pour des optimisations de calcul chirurgicales et l'analyse de sécurité.
